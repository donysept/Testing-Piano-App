# Contributing to PianoKu

Terima kasih sudah tertarik untuk kontribusi ke PianoKu! 🎉

## Getting Started

1. Fork repository ini
2. Clone fork Anda: `git clone https://github.com/YOUR_USERNAME/Testing-Piano-App.git`
3. Buat branch feature: `git checkout -b feature/AmazingFeature`
4. Commit changes: `git commit -m 'Add some AmazingFeature'`
5. Push ke branch: `git push origin feature/AmazingFeature`
6. Buka Pull Request

## Code Style

- Gunakan Kotlin style guide
- Proper naming conventions untuk variables, functions, dan classes
- Add comments untuk logic yang kompleks
- Format code dengan Ctrl+Alt+L di Android Studio

## Commit Messages

Gunakan format yang jelas:
- `feat: add new feature`
- `fix: fix bug description`
- `docs: update documentation`
- `style: formatting changes`
- `refactor: code refactoring`
- `test: add tests`

## Pull Request Process

1. Update README.md dengan details changes jika perlu
2. Pastikan code sudah tested
3. PR harus punya clear title dan description
4. Link ke related issues jika ada

## Issues

Sebelum membuat issue baru:
1. Search dulu existing issues
2. Provide clear description
3. Include reproducing steps jika bug report
4. Attach screenshots jika relevan
